调用方法：

new Window().alert(selector,{options});

options参数

参数 默认值 说明
width 200 窗体宽度
height 200 窗体高度
content “” 窗体内容
title '系统消息' 标题
hasCloseBtn false 是否有关闭按钮
text4AlertBtn '' alert确定按钮文字
hasMask true 是否有遮罩层
hasCloseBtn false 是否有关闭按钮
text4ConfirmBtn '确定' confirm确定按钮文字
text4CancelBtn '取消' confirm取消按钮文字

events事件

名称 调用方式 说明
hander4AlertBtn new Window().alert(selector,{hander4AlertBtn:function(){ }}); alert点击确认按钮后callback函数
handler4CloseBtn new Window().alert(selector,{handler4CloseBtn:function(){ }}); alert点击取消按钮后callback函数
handler4ConfirmBtn new Window().confirm(selector,{handler4ConfirmBtn:function(){ }}); confirm点击确认按钮后callback函数
handler4CancelBtn new Window().confirm(selector,{handler4CancelBtn:function(){ }}); confirm点击取消按钮后callback函数
